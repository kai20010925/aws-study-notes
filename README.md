# AWS学習メモ

## 概要

このリポジトリは、未経験からクラウドインフラエンジニアを目指すために、AWSの基礎を整理した学習メモです。

現在はAWS Certified Solutions Architect - Associate（SAA）の学習を進めながら、主要サービスの役割や試験で問われやすい考え方を自分の言葉でまとめています。

## 主な役割

- AWSの基本サービスを復習しやすくする
- サービスごとの役割や違いを整理する
- SAAでよく出るポイントを確認できるようにする
- 面接で「何を学んだか」を説明しやすくする

## SAAでよく出るポイント

SAAでは、単にサービス名を覚えるだけでなく、要件に対してどのサービスを選ぶかがよく問われると感じています。

そのため、可用性、セキュリティ、スケーラビリティ、コスト、運用負荷を意識しながら学習しています。

## 学んだこと

AWSはサービス数が多いですが、まずはネットワーク、コンピューティング、ストレージ、セキュリティの基本を押さえることが大事だと感じました。

特にVPC、EC2、IAM、S3は他のサービスを理解する土台になるため、優先して整理しています。

## 学習目的

- クラウドインフラの基本的な仕組みを理解する
- AWSの主要サービスを自分の言葉で説明できるようにする
- SAAの試験対策として重要ポイントを整理する
- 将来的にインフラ構築や運用の仕事に関われるようにする
